## 快速开始（3步）

**触发词**：`帮我优化提示词` / `改进我的Prompt` / `提示词蒸馏` / `prompt alchemist`

**或者直接粘贴**：任何你想优化的提示词文本（无需指令，自动进入蒸馏流程）

### 第一步：粘贴原始提示词

```
帮我分析一下最近的科技趋势
```

### 第二步：AI 给出诊断 + 优化成品

```
🔍 诊断报告
- 任务类型：分析报告
- 核心问题：角色堆砌风险、维度不明确、无格式约束
- 优化方向：从开放到结构化

⚡ 蒸馏成品（可直接复制使用）
<role>科技行业分析师，专注新兴技术趋势与商业影响评估</role>
<task>从技术突破/商业影响/竞争格局/风险挑战四个维度分析...</task>
<format>
| 趋势 | 技术亮点 | 商业价值 | 风险 | 推荐关注度 |
...
</format>
```

### 第三步：拿去用 / 追问微调

- `再压缩一下` → 执行增量压缩
- `换个风格` → 调整语气/受众
- `我不认同第X点` → 回退该变更

## 复杂度分级输出

| 任务类型 | 你的提示词特征 | 输出模块 |
|---------|--------------|---------|
| **简单** | <100字、单一问题、明确需求 | 诊断（3-5行）+ 成品 |
| **中等** | 需要结构调整或多步骤 | 诊断 + 结构重构 + 成品 |
| **复杂** | 涉及推理/决策/创意、需求模糊 | 诊断 + 结构 + 思考增强 + 成品 + 变更说明 |

> ⚠️ 不会对简单任务过度设计。如果诊断结果只有1个小问题，只输出诊断+成品并注明"微小优化"。

## 核心能力

### 1. 四步蒸馏流程

```
🔍 诊断  →  诊断原始提示词的问题（象限定位/反模式识别/模型适配）
🏗️ 结构  →  搭建提示词骨架（XML标签分层/变量占位/格式锚定）
🧠 思考  →  嵌入自适应思考深度（简单→复杂分级引导，中文优化）
⚡ 压缩  →  8步操作清单压缩（约束保全检查，一行测试，共振验证）
```

### 2. 多轮对话蒸馏

支持对话历史场景的增量优化：

```
第1轮：帮我写一个产品文案
第2轮：换成年轻化风格
第3轮：再加一个情感共鸣的结尾
→ AI 自动识别：角色延续/风格叠加/约束演进 → 输出第3轮增量成品
```

### 3. 模型适配

支持指定目标模型（DeepSeek / GPT / Claude / Gemini），自动调整提示词风格和约束策略。

### 4. 反模式防御

内置 18 个反模式（14个单轮 + 4个多轮） + 5个核心失效模式，开箱即用。
